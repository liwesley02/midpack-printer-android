# Brother Print SDK

Download the Brother Print SDK for Android from:
https://support.brother.com/g/s/es/dev/en/mobilesdk/android/index.html

Required file: `BrotherPrintLibrary.aar`

Place the AAR file in this directory before building the Android app.

For QL-820NWB printer, you'll need:
- Brother Print SDK v4.x or later
- Supports Bluetooth connectivity
- Label printer functionality
